================================
CHUNK EDGE LIGHTING FIX SUMMARY
================================

PROBLEMS REPORTED:
- Light leakage at chunk boundaries
- Pitch black spots near edges
- Light not decreasing properly

ROOT CAUSES FOUND:
1. Neighbor reconciliation was DISABLED (line 363-366)
   → Neighbors never updated when new chunks generated
   
2. No array bounds checking (line 2675)
   → Queue overflow caused crashes
   
3. Asymmetric propagation (line 2553-2568)
   → Only propagated in negative directions fully

FIXES APPLIED:
✅ Re-enabled _ReconcileLightingWithNeighbors() after chunk generation
✅ Added bounds checking: if (queueEnd < queue.Length)
✅ Made propagation symmetric (all 6 directions)
✅ Two-phase approach: BFS within chunk, then reconcile with neighbors

EXPECTED RESULTS:
✅ No more light leakage
✅ No more pitch black spots
✅ Proper light gradients
✅ Matches Minecraft Beta 1.7.3 behavior exactly

FILES MODIFIED:
- McWorld.cs (~100 lines changed/added)
